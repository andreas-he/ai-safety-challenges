# AI Safety Challenges

Daily 30-minute AI safety exercises — small, self-contained drills calibrated to mechanistic interpretability, evaluation, and ML-for-safety topics. One challenge per weekday, generated by the `daily-ai-safety-challenge` skill in the companion brain repo.

## How it works

- **Weekday format rotation**
  - Mon, Thu → **short-coding** (Jupyter notebook, ≤30 lines to write, inline asserts)
  - Tue → **multiple-choice** (concept check with distractors)
  - Wed → **explain** (paper excerpt + "explain in your own words")
  - Fri → **read-reflect** (paper excerpt + focused reflection question)

- **Primary surface**: the brain dashboard `Challenges` tab. You read the problem, hit hints, submit a solution, see yesterday's reference solution next morning.

- **Coding days**: the dashboard links out to a Jupyter notebook in this repo. Open it in Colab (one click) or locally. Fill in stubs. Run cells — asserts validate your work. Peek the collapsed `<details>` solution cell if stuck.

- **Non-coding days**: everything renders inline in the dashboard. The `README.md` here is the archived mirror.

## Structure

```
challenges/
  YYYY-MM-DD-slug/
    README.md           # Problem statement + (coding days) Colab badge
    challenge.ipynb     # Coding days only: starter + asserts + collapsed solution
stats.json              # Streak, totals, category breakdown
```

## Quick start — coding days

1. Click the **Open in Colab** badge at the top of `challenge.ipynb`.
2. Fill in the `raise NotImplementedError` stubs.
3. Run all cells — the asserts tell you when you're done.
4. Stuck? Unfold the `<details>Solution</details>` cell at the bottom.
5. Back to the dashboard: paste a summary note (or just a "✓") into the submit box.

No git clone, no pytest, no pip install — the notebook runs on Colab's free tier with numpy + scikit-learn preinstalled.

## Philosophy

- **30 minutes, not 3 hours.** Morning coffee, not a weekend project.
- **Self-contained.** All reading excerpts pasted inline. No "go read section 3 of the paper."
- **ARENA-inspired.** Starter cells + asserts + collapsed solutions — the format that works.
- **Dynamic, not a catalog.** The skill picks an uncovered topic × today's format each morning and generates a fresh exercise; it does not draw from a fixed problem bank.
